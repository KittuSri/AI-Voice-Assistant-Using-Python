# AI Voice Assistant Using Python

## Overview
This project is a voice-controlled AI assistant developed using Python. It recognizes voice commands, opens websites, plays music, and answers user questions using a local Large Language Model (Ollama + Llama 3.2).

## Features

- Wake word detection ("Jarvis")
- Voice recognition
- Text-to-Speech
- Open Google, YouTube, Facebook
- Play songs
- AI-powered question answering
- Offline AI using Ollama
- Exit command

## Technologies

- Python
- SpeechRecognition
- pyttsx3
- Ollama
- Llama 3.2
